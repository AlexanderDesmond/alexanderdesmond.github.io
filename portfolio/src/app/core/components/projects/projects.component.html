Gwent Chest

Gwent Chest is an online deckbuilder and deck catalogue for the digital collectible card game Gwent. Users can browse a catalogue of decks other users have submitted to find new ideas for their own decks. Or users can create their own decks and save them on Gwent Chest so others users can try it in-game.

Angular is used for the front-end, with Node.js and Express.js being used in the back-end. Authentication is handled by JSON Web Tokens (JWT) with Bcrypt being utilised to perform the hashing and salting of passwords. MongoDB is used as the database, with Mongoose and Joi being used for defining the structure and validating the JSON data before it reaches the database.

ImageBoardy McImageBoardFace

A web application which allows users to create and reply to posts; except all posts and comments must be images. So, basically reddit except exclusively for images.

The front-end was created with Angular, while Node.js and Express.js were used in the back-end. MongoDB was used for the database.

Nutty Friends!

Nutty Friends! is a story-based mobile game which utilises augmented reality (AR) to raise awareness of the nutritional benefits of nuts and encourage young people to incorporate them into their diets.

The game was developed in Unity with Vuforia selected as the AR framework.
